Readable sidenotes without layout scripts

The note starts in the reading flow, then moves into the margin when anchor positioning is available 1.

1. Unsupported and narrow layouts keep this note inline, so the annotation remains readable.

That fallback-first structure makes the modern layout a progressive enhancement instead of a hard dependency.

  • The marker receives anchor-name; the note stays in source order.
  • Wide supported layouts use position-anchor and anchor() to align the margin note.
  • Narrow or unsupported layouts keep the note inline with the paragraph.
