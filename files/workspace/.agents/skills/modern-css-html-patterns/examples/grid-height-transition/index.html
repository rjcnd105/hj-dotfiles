Dynamic disclosure height

Grid height transition

The panel animates between collapsed and natural content height by transitioning one grid row from 0fr to 1fr. No arbitrary max-height value is needed.

Content keeps its natural size

The outer grid controls the track. The inner wrapper clips the content while the track grows and shrinks.

  • Use grid-template-rows: 0fr for the collapsed state.
  • Use grid-template-rows: 1fr for the expanded state.
  • Remove the transition for reduced-motion users.
No magic max The timing no longer depends on guessed content height.
One inner wrapper The child handles overflow while the grid track animates.
Instant fallback Without transition support, the panel still opens and closes.
