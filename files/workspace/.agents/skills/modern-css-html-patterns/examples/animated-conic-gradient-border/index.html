Animated conic gradient border

A pseudo-element sits behind the card face and paints a conic gradient. A registered angle custom property rotates the color stops smoothly.

  • The first color stop repeats at the end for a clean loop.
  • The blurred duplicate layer provides glow without extra markup.
  • Reduced-motion users keep the static gradient border.
